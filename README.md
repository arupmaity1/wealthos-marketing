# WealthOS — Marketing micro-site

Static marketing site for WealthOS (the wealth-management & family-office platform).
Served by nginx via the included Dockerfile. Deployed on Coolify at https://get.wealthos.steerfry.com.

Pages: `index.html` (home), `license.html` (source-available license). Styles in `styles.css`;
product screenshots in `assets/shots/`.
